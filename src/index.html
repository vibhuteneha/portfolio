Timer Challenge Game

This Timer Challenge Game was built using React.js. It lets you test your reaction time. You can start any of the timers to start the game and then click the Stop Challenge button to stop the timer. It will show you the time you took to click the button. The less time you take, the higher your score. If you take more than the associated time to click the stop button, it will show you a message that you lost the game.
Technologies used: React.js, JavaScript, HTML, CSS.
